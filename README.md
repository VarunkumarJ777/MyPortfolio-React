# My Portfolio (React + TypeScript)

Simple React + Vite + TypeScript starter for a personal portfolio. Update the content in `src/App.tsx` to make it yours.

## Run locally

1. Install dependencies
   ```bash
   npm install
   ```
2. Start dev server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Notes
- Vite is configured in `vite.config.js`.
- Global styles live in `src/index.css`; component styles in `src/App.css`.
- Update the contact link in the hero to your email or another CTA in `src/App.tsx`.
